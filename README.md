# Shamsa Batool — Portfolio

Professional portfolio website built with **Next.js 14** showcasing my work as an AI/ML Engineer & Agentic AI Developer.

## 🚀 Live Website

> _Coming soon_ — Deploy on Vercel or GitHub Pages

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** CSS Modules + Vanilla CSS
- **Font:** Inter (Google Fonts)
- **Deployment:** Vercel-ready

## 📁 Project Structure

```
├── app/
│   ├── layout.js          # Root layout with SEO metadata
│   ├── page.js            # Main page assembling all sections
│   └── globals.css        # Design system & global styles
├── components/
│   ├── Navbar/            # Sticky nav with mobile drawer
│   ├── Hero/              # Hero section with headshot & stats
│   ├── About/             # Highlight cards
│   ├── Experience/        # Timeline-style work history
│   ├── Projects/          # Filterable project grid
│   ├── Skills/            # Categorized skill badges
│   ├── Education/         # Degrees, certs, honours
│   ├── Contact/           # Social links & CTA
│   └── Footer/            # Minimal footer
├── public/
│   ├── images/            # Headshot & assets
│   └── resume/            # Downloadable resume PDF
```

## 🏃 Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
